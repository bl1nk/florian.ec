Braincrafted is a collection of carefully crafted open source libraries by Florian Eckerstorfer.

  • BootstrapBundle
    BraincraftedBootstrapBundle integrates Bootstrap into Symfony2 by providing templates, Twig extensions, services and commands.
    Github
  • Symfony2CSBundle
    Automatically install Symfony2 coding standard for PHP_CodeSniffer.
    Github
  • TestingBundle
    Tools for testing Symfony2 applications
    Github
  • ArrayQuery
    ArrayQuery is a library to query arrays.
    Github
  • Json
    Object-orientated wrapper for json_encode() and json_decode() with error handling.
    Github
  • ExpExp
    Expands expressions. Kinda the opposite of what regular expressions do.
    Github
