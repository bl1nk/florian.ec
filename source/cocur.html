Cocur is a collection of carefully crafted open source libraries by Florian Eckerstorfer.

  • Slugify
    Converts a string to a slug. Includes integrations for Symfony, Silex, Laraval and Twig.
    Github API documentation
  • BackgroundProcess
    Start processes in the background that continue running when the PHP process exists.
    Github
  • HumanDate
    Transforms dates into a human-readable form.
    Github API documentation
  • NQM
    Named Query Manager (NQM) helps you SQL organise queries in files.
    Github API documentation
  • Watchman
    PHP wrapper for facebook/watchman.
    Github
